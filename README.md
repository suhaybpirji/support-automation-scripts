# 🛠️ Support Automation Scripts

A collection of **PowerShell** and **Bash** scripts designed for real-world 2nd Line Support, Cloud Support, and Infrastructure troubleshooting.

This repository demonstrates automation skills across Windows Server, Active Directory, Linux, and general system administration.

---

## 📂 Folder Structure

support-automation-scripts/
│
├── powershell/     # Windows & AD automation scripts
├── bash/           # Linux automation scripts
└── README.md

---

## 🔵 PowerShell Scripts (Windows / AD)

These scripts cover common 2nd Line tasks:

- Bulk create Active Directory users  
- Reset AD passwords  
- Unlock AD accounts  
- Export Windows Event Logs  
- Restart Windows services  
- Check disk usage  
- Get installed software  
- Network adapter information  

---

## 🟢 Bash Scripts (Linux)

These scripts demonstrate Linux troubleshooting and automation:

- Check disk space  
- Monitor CPU usage  
- Backup directories  
- Analyse failed login attempts  
- Restart services  
- Check open ports  
- Monitor memory usage
